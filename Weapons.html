STINGER.

SPECTRE.

ประเภท // ปืนกล SMG นี้มีศักยภาพในระยะกลางถึงไกลมากกว่าปืนอื่นที่คล้ายกัน แต่อัตราการยิงและความคล่องตัวก็น้อยลง อาวุธที่มีดีในทุกด้าน มีความสมดุลของความเสียหาย อัตราการยิง และความแม่นยำที่ดี ทั้งในระยะใกล้และกลาง

BULLDOG.

ประเภท // ปืนไรเฟิล สัตว์ร้ายที่ยิงได้ผลแน่นอนหากคุณมีโอกาสได้เลือกการยิงเอง โหมดการยิงเสริมทำให้คุณได้ ADS และละทิ้งความแม่นยำ เบิร์ส ให้แก่ใครก็ตามที่จะลองดีกับคุณในระยะกลางไปจนถึงระยะไกล

ประเภท // ปืนไรเฟิล ยิงแบบอัตโนมัติจัดเต็มเมื่อมีคนมาทดสอบคุณใกล้ ๆ หรือยิงเบิร์สแบบรัดกุมเพื่อทำให้ศัตรูทุกรายตะเกียกตะกาย ใช้งานได้ดีที่สุดเมื่อยืนอยู่กับที่
